Timeago

Fast and lightweight

Date and time package for your Go project that converts given date into 'n time ago' format.

Get Started →
What is Timeago?
🧪

Well tested

You cannot find datetime package that is tested better than this one. It connected to several code quality analyzers.

🌍

Multilingualism

You can output date in several different languages. You can easily contribute any language that you want in simple 4 steps.

🧩

Flexible

Timeago can parse several different date time formats, including UNIX timestamp, date string and time package.
